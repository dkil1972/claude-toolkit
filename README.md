# Claude Toolkit

Skills and tools for Claude Code, focused on software development workflows.

## Skills

| Skill | Description |
|-------|-------------|
| `discover` | Investigate a reported problem, diagnose root causes, audit for related issues, and produce structured spec files |

## Usage

Skills can be added to Claude Code via the `/install-skill` command or by copying the skill files into `~/.claude/skills/`.
